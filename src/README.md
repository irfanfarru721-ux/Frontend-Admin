Admin Frontend (React + Vite)
----------------------------

How to run locally:
1. npm install
2. create .env with:
   VITE_ADMIN_API=http://localhost:5001/api/admin
   VITE_CLOUDINARY_UPLOAD_URL=https://api.cloudinary.com/v1_1/<cloud_name>/upload
   VITE_CLOUDINARY_UPLOAD_PRESET=<unsigned_preset>
3. npm run dev
